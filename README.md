# Inventory Managment System

A simple inventory management system for general use.

## Features
- Login/Register
- Change Password
- Create/Update/Delete Products

## Tech Stack
- PHP (Vanilla)
- MySQL
- PDO (Prepared Statements)
- Apache (XAMPP)
- Postman (API Testing)

## Architecture
- MVC
- REST
- JSON

## What I learned
- Designing REST API
- Switched from using MySQLi to PDO
- Proper use of MVC structure
- Handling JSON request and response
